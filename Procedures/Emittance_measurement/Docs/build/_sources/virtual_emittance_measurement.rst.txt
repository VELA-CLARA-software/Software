Virtual Emittance measurement Class
===========================
This page contains the description of the BeamSizeDetermination class for the Virtual Emittance measurement.
It takes a set of currents in order to determine the beam sizes at the observation point using the Virtual 
accelerator (daughter class of GeneralQuadScan :ref:`Emittance_Quad_Scan`.
