GeneralQuadScan (parent class)¶

This page contains the description of the General Quad Scan Class (parent class).

class QuadScan.QuadScanclass.GeneralQuadScan[source]¶

Parent class that has all the methods to measure the emittance values by reconstructing the covariance matrix at reconstruction point from the beamsizes and optimised quad strengths

Variables:
  • charge – Charge: 10 pC
  • SMatrix – Antisymmetric matrix: Smatrix
  • rel_gamma – Relativistic gamma: 30.0 / m_e_MeV
  • qlength – Quad length: 0.1007
  • dlen1 – Drift length: 0.105327
  • dlen2 – Drift length: 0.299300
  • dlen3 – Drift length: 1.033333
  • cov_matrix – Calculated covariance matrix at reconstruction point: np.zeros((4, 4))
  • emits – Emittances of normal modes: np.zeros((2, 1))
  • emits_matrix – Emittances matrix transformation:[]
  • qstrengths – Optimised quad strengths: []
  • Nsteps – Number of steps of quad scan: 20
  • input_dist – path of the input distribution for simulation: ‘’
  • d_matrix_step – D matrix calculated for a single set of quad values: np.zeros((3,10))
  • d_matrix – total D matrix over all quad scans: np.array([])
  • beam_sizes – data obtained from observation point used to calculate the emittance: OrderedDict()
  • screen_data – Data ordered obtained from observation point in order to find the cov. matrix: np.array((3*nstep,10))
  • path_output – Path to store the plot of the quad scan: ‘’
