Example Scripts
===============
This page contains the example scripts for the CATAP python library.

Each script will demonstrate how to get values from all, multiple, and single 
hardware objects either using the Factory class or the specific hardware class
itself.

Please refer to the :doc:`./HardwareTypes` and :doc:`./Factories` documentation
pages for further information on setting/monitoring values.
